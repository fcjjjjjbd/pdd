# 示例图片目录

此目录包含二手交易平台的示例图片，用于展示商品卡片组件的效果。

## 图片列表

- phone.jpg - 手机示例图片
- laptop.jpg - 笔记本电脑示例图片
- headphone.jpg - 耳机示例图片
- shoes.jpg - 鞋子示例图片

## 使用说明

这些图片仅用于开发和测试阶段，实际部署时应替换为真实的商品图片。在商品数据中，图片路径应设置为：

```
/static/images/demo/图片名称.jpg
```

例如：

```javascript
image: '/static/images/demo/phone.jpg'
```
